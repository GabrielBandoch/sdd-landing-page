# TravelX Landing Page

Uma landing page responsiva e moderna desenvolvida para a agência de viagens "TravelX", projetada para converter visitantes em clientes que buscam roteiros personalizados.

## Processo Spec-Driven Development (SDD)

Este projeto foi construído utilizando a metodologia Spec-Driven Development. O fluxo de trabalho consistiu em:

1. **Especificação (Spec)**: Extração dos requisitos visuais e funcionais a partir do protótipo Stitch. Elaboração de um Product Requirements Document (PRD), Especificação Técnica e Task Breakdown detalhado.
2. **Planejamento**: Divisão do projeto em 5 fases lógicas (Setup, Assets, Componentes, Integração, Documentação).
3. **Implementação**: Construção da interface com foco em mobile-first, utilizando componentes React granulares e classes utilitárias do Tailwind CSS. 
4. **Validação**: Verificação dos requisitos originais (design, acessibilidade e responsividade) e testes em diferentes resoluções.

## Tecnologias Empregadas

- **React 19** + **TypeScript**: Construção da interface do usuário de forma declarativa e com tipagem estática robusta.
- **Tailwind CSS v4**: Framework CSS utility-first para estilização rápida, coerente e com performance otimizada.
- **Vite**: Bundler ultra-rápido para desenvolvimento e build.
- **Lucide React**: Biblioteca de ícones moderna e leve.

## Como Executar Localmente

Siga os passos abaixo para testar o projeto no seu ambiente local:

1. Acesse o diretório do projeto:
   ```bash
   cd travelx
   ```
2. Instale as dependências:
   ```bash
   npm install
   ```
3. Inicie o servidor de desenvolvimento:
   ```bash
   npm run dev
   ```
4. Abra o navegador no endereço indicado no terminal (normalmente `http://localhost:5173`).
